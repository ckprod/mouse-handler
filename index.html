mouse-handler

Base class to distinguish between left mouse button actions: click vs. drag

Repository and description on github

Sample

Options: distance: 20
This sample shows an arbitrary usage of the mouse-handler template. The minimum drag distance is set to 20 pixel. Within a circle with this radius any mouse movement won't count as drag and triggers the click action on left mouse button release.
